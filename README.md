# Tutorial Management System 
A backend CRUD application built using Spring Boot, Spring Data JPA, H2 Database and JUnit.  
This project shows REST APIs for managing Tutorials.

---

## Features

- CRUD operations for Tutorial entity
- REST API using Spring Boot
- H2 inmemory database 
- Layered architecture (Controller - Service - Repository)

---

## Technologies Used
- Spring Boot
- Spring Data JPA
- Hibernate ORM
- H2 Database
